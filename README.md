# Solara-Panel
the apex of Roblox Admin Panels.

how to setup:

this is the V1 of the Solara Panel. expect bugs, and expect the V2 to be out soon. or not soon. anyway,
to setup, firstly you need to change your user in SolaraCore. 
then make sure API and HTTPS settings in security are on,
once you made these steps you can use the panel. enjoy!
